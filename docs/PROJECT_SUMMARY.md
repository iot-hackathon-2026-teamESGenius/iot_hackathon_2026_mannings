## 📋 项目概述

**项目名称**: Mannings 门店自提 SLA 优化系统  
**参赛组别**: 学生组  
**服务客户**: DFI Retail Group - Mannings (万宁)  
**开发团队**: Team ESGenius

### 核心目标
为香港连锁药店 Mannings 开发端到端的**门店取货服务等级协议(SLA)优化系统**，通过 AI/ML 预测和数学优化算法提升顾客取货体验并控制运营成本。

## 🎯 评审标准评分对照

### A. Innovation & Creativity (创新与创意) - 25%

#### 1. 创意的独特性与创新性 ⭐⭐⭐⭐⭐

**端到端耦合优化架构**
| 传统方法 | 本系统创新方法 |
|---------|---------------|
| 预测 → 固定数值 | 预测 → 置信区间驱动决策 |
| 库存独立优化 | 库存与路径约束联动 |
| 路径使用单一需求 | 路径使用需求情景分析 |
| SLA事后估算 | SLA集成到优化目标 |

**四大创新点**:
1. **预测-决策闭环**: 预测不确定性直接驱动调度决策
2. **多情景鲁棒优化**: 基于Min-Max策略的最坏情况优化
3. **学习增强策略**: 基于特征评分的需求缩放
4. **AI智能助手**: 集成AWS Bedrock Claude提供决策建议

#### 2. 创造性的问题解决方法 ⭐⭐⭐⭐⭐

**鲁棒路径优化算法**:
```python
# 从预测置信区间生成需求情景
scenarios = scenario_generator.generate_from_forecasts(
    forecasts=demand_forecasts,
    num_scenarios=5,
    method="quantile"  # P10/P50/P90分位数
)

# Min-Max鲁棒策略选择
robust_solution = optimizer.robust_optimization(
    scenarios=scenarios,
    selection_criterion="min_max_distance"  # 优化最坏情景
)
```

**智能加权预测**:
- 工作日/周末自动切换预测策略
- 节假日效应自动检测与调整
- 天气因素实时融合

#### 3. 新型运营模式 ⭐⭐⭐⭐

- **实时调度优化**: 支持动态添加/删除调度任务
- **可视化决策支持**: 地图实时显示路线与门店标记
- **AI辅助决策**: 自然语言交互获取优化建议

## 📊 功能模块详情

### 1. 需求预测模块
- **算法**: Facebook Prophet 时序预测
- **输出**: P10/P50/P90 置信区间
- **特性**: 节假日效应、天气融合、促销调整

### 2. 路径优化模块
- **算法**: OR-Tools CVRPTW
- **创新**: 多情景鲁棒优化 (Min-Max)
- **集成**: 高德地图真实道路路径

### 3. SLA预测模块
- **方法**: 概率预测模型
- **输出**: SLA达成概率、风险评分
- **建议**: 改进措施推荐

### 4. 调度管理模块
- **功能**: 增删改查调度任务
- **可视化**: 地图路线、门店标记
- **持久化**: 本地存储刷新保持

### 5. AI智能助手
- **模型**: AWS Bedrock Claude 3
- **能力**: 自然语言决策建议
- **场景**: 异常诊断、优化建议
